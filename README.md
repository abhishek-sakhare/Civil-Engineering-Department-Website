# Official Department Website – Civil Engineering, NIT Agartala

The official website of the Civil Engineering Department, NIT Agartala, developed to provide students, faculty, researchers, and visitors with easy access to departmental information, announcements, academic resources, and activities.

## Features

- Responsive design for desktop, tablet, and mobile devices
- User-friendly navigation and accessible interface
- Department information, academics, faculty, research, and activities sections
- Organized and maintainable content structure
- Integrated with the institute's main website

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Overview

This project was developed as the official departmental website of the Civil Engineering Department, NIT Agartala. The focus was on creating a clean, responsive, and accessible platform that effectively presents departmental information while ensuring a smooth user experience across different devices.
